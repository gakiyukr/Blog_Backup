## 准备

1. Alipay HK（废话）
2. 带3DS的内地外币卡（只能Visa/Mastcard）

## 效果

可以在Alipay HK绑定内地外币卡，正常在Play商店、以及其他支持Alipay+的商户消费，但是无法在内地商户消费，如淘宝（外卡内抛）整体来说意义不大。~~（绑上去图一乐呵）~~

- 建议绑定无限返现的卡。
- 可以多试几次，毕竟这算是个小BUG，一次不一定成功。
- ~~（别傻了吧唧的去问客服为什么绑不上去）~~
- 本方法来源于卡粉群（[@DocOfCard](https://t.me/DocOfCard/238)）个人只做完善讲解
